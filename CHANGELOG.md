### 经验积累

- **经验 001**: Windows UTF-8 编码问题处理
  - 问题：Read/Bash 工具处理中文文件时中断
  - 解决：使用 Python 代替 Bash，显式指定 encoding='utf-8'
  - 配置：`AI Collaboration Rules` → `6. Windows UTF-8 编码处理`

- **经验 002**: 工作目录管理
  - 问题：在子目录执行文件操作导致路径错误
  - 解决：执行前确认当前目录，使用绝对路径或正确切换
  - 配置：`AI Collaboration Rules` → `7. 工作目录管理`
